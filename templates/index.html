END POINTS

  • GET "/professors" will return a list of all professors
  • GET "/dinners" will return a list of all dinners
  • GET "/students" will return a list of all students
  • GET "/professors/[string:uniqueID]" will return a specific professor by Unique ID
  • GET "/student/[string:netID]" will return a specific student by netID
  • GET "/dinner/[int:id]" will return a specific dinner by id
